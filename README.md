# 🍼 Todaktalk - 또닥톡\n\n**또닥톡**은 자녀의 성격, 관심사, 지역을 기반으로  \n가장 잘 어울릴 친구를 찾아주는 **맞춤형 매칭 플랫폼**입니다.  \nAI 추천 시스템, 챗봇 상담, 지도 기반 시각화 기능까지 탑재된  \n부모와 아이 모두를 위한 스마트한 육아 커뮤니티 앱입니다.\n\n---\n\n## ✨ 주요 기능\n\n- 🔐 **소셜 로그인 (카카오 / 네이버)**  \n  간편한 인증을 통해 부모 정보 등록, 자녀 정보 연동\n\n- 👶 **자녀 프로필 등록**  \n  최대 5명까지 등록 가능 / 생년월일, 성별, 성격, 지역, 선호 활동 포함\n\n- 🧠 **친구 매칭 알고리즘 (Flask API)**  \n  자녀의 성격, 관심사, 연령, 지역 기반으로 유사한 친구 추천\n\n- 🗺️ **지도 기반 Playground 분포 시각화**  \n  Kakao Maps SDK를 활용해 우리 동네 아이 분포 및 놀이공간 시각화\n\n- 🤖 **RAG 기반 챗봇 상담 (FastAPI + LLM)**  \n  인근 시설 추천(TF-IDF), 육아 질문 응답(OpenAI 기반)\n\n- ⏰ **하원시간 및 예방접종 알림 기능**  \n  자녀별 하원시간/접종일 설정 → 알림 등록 가능\n\n- 📱 **상단 고정 TopView UI 구성**  \n  로고, 알림 아이콘, 햄버거 메뉴 항상 상단 고정 노출\n\n---\n\n## 🛠 기술 스택\n\n| 구분         | 기술 구성                                   |\n|--------------|--------------------------------------------|\n| 프론트엔드   | `SwiftUI`, `UIKit`, `UIViewControllerRepresentable` |\n| 지도 API     | `Kakao Maps SDK (Native)`                  |\n| 백엔드       | `Flask` (친구 매칭), `FastAPI` (챗봇)        |\n| 데이터베이스 | `Firebase Firestore`, `MySQL`               |\n| AI 모델      | `OpenAI API`, `TF-IDF`, `LangChain` 기반 RAG |\n| 챗봇 UI      | `Streamlit`, `SwiftUI ChatbotView`          |\n| 로그 수집    | `AWS API Gateway`, `Lambda` (플레이 클릭 로그) |\n\n---\n\n## 📂 폴더 구조\n\n```plaintext\nTodaktalk/\n├── App/\n│   ├── LandingView.swift\n│   ├── LoginView.swift\n│   ├── SignUpFormView.swift\n│   ├── ChildFormView.swift\n│   ├── SelfInfoView.swift\n│   └── ...\n├── Match/\n│   ├── match_algorithm.py\n│   └── app.py\n├── Chatbot/\n│   ├── rag_chat.py\n│   └── streamlit_ui.py\n├── Map/\n│   ├── PlaygroundMapViewController.swift\n│   └── PlaygroundWrapperView.swift\n└── Resources/\n    └── Assets.xcassets\n```\n\n---\n\n### 🛠 커스터마이징 Tip:\n\n- Kakao/Google Map API Key, Firebase 설정 등은 `.env` 혹은 `Config.swift`로 따로 관리하세요.\n- 챗봇/매칭 백엔드는 `requirements.txt`와 `Procfile`을 이용해 배포할 수 있습니다 (예: Render, Railway 등).\n\n---\n\n## 🚀 실행 방법\n\n### iOS 앱 실행\n\n```bash\n# CocoaPods 설치\npod install\n\n# Xcode에서 .xcworkspace 열기 후 실행\nopen Todaktalk.xcworkspace\n```\n\n### 친구 매칭 서버 실행 (Flask)\n\n```bash\ncd Match\npip install -r requirements.txt\npython app.py\n```\n\n### 챗봇 서버 실행 (FastAPI)\n\n```bash\ncd Chatbot\npip install -r requirements.txt\nuvicorn rag_chat:app --reload\n```\n\n---\n\n## 📌 개발 로드맵\n\n- [x] 카카오/네이버 소셜 로그인
